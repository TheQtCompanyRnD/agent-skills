# Contributing

See [`CONTRIBUTING.md`](https://github.com/TheQtCompanyRnD/agent-skills/blob/dev/CONTRIBUTING.md)
in the repo for the full cross-platform packaging story.

The canonical format is `SKILL.md` with YAML frontmatter in a
directory-based structure — this works natively on Claude Code and
Codex CLI, and can be adapted to other platforms through condensed
variants.
